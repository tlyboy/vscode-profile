# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

这是一个 VS Code 配置文件仓库，用于存储和共享个人 VS Code 配置（Guany's VS Code profile）。

## 项目结构

- `Default.code-profile` - VS Code 导出的配置文件，包含编辑器设置、快捷键、已安装扩展列表等
- `.vscode/` - 仓库级别的 VS Code 配置
- `.editorconfig` - 编辑器通用配置（2空格缩进、UTF-8、LF换行）
- `.prettierrc` - Prettier 配置（无分号、单引号）

## 代码风格

- 使用 Prettier 格式化代码
- 2 空格缩进
- 单引号
- 不使用分号
- 换行符使用 LF
- 文件末尾保留空行
